# Specification

## Summary
**Goal:** Inform the user that the requested Android browser application cannot be built on this platform.

**Planned changes:**
- None

**User-visible outcome:** The user understands that this platform is limited to web applications with Motoko backend and React frontend, and cannot create Android APKs, native mobile applications, browser engines, or browser extensions. Native mobile features like system-level sandboxing, VPN configuration, background media playback, and download management are not supported.
